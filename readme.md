#### Working Locally

To work on this project locally, you'll need a copy of [Hugo](http://gohugo.io/).

```sh
git clone git@github.com:putterschool/putterschool.com.git
cd putterschool.com
hugo server
```

#### Deployment

The `master` branch is automatically deployed. Magic!
